# 👩🏻‍💻 Helen Jerônimo

**`Análise de dados`**

Me chamo Helen Jerônimo, tenho 24 anos e sou de São Paulo. Me formei em Administração Pública pela Fundação Getúlio Vargas (FGV). Atualmente, estou cursando Ciência de Dados na Anhanguera.
